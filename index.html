Light and Sound Memory Game
Welcome to the game! Repeat back the pattern by pressing the corresponding buttons, but make sure to not forget the pattern! Press start to begin.
Generate a new random color pattern for your next round, otherwise the same will be used each round so that you can retry the same pattern.
  • Select the length of the color pattern, this contributes to a longer and more difficult round.
  • Select the amount of mistakes your next round will allow. Select 0 if you wish to turn off the mistakes feature.
The faster the countdown clock, the less time you have. The timer resets after each guess; you can also choose to disable the feature entirely.
